Абстрактный интерфейс генератора псевдослучайных чисел.

МетодОписание

class LCGGenerator

Линейный конгруэнтный генератор: Xn+1 = (a·Xn + c) mod m

a = 1'103'515'245,  c = 12'345,  m = 2³¹

next(min, max) — нормализация через state % span

class VonNeumannGenerator

Метод середины квадрата с константой Вейля.

X = middle(X² + w),  w += 0x9E3779B97F4A7C15

Предотвращает вырождение в ноль при длительной генерации.

class XorshiftGenerator

Сдвиговый генератор Marsaglia (Xorshift32).

state ^= state << 13;
state ^= state >> 17;
state ^= state << 5;

class MT19937Generator

Обёртка над std::mt19937 для сравнительного бенчмарка.

chiSquareUniformityTest(data, min, max, bins=10)

Критерий Хи-квадрат на равномерность распределения.

data — выборка целых значений

bins — число карманов K (df = K−1)

Chi2Result — chi2, p_value, critical_value, passed, verdict()

Вердикт: chi2 ≤ critical и p ≥ 0.05 → ПРИНИМАЕТСЯ
