# Лабораторная работа: Генерация и тестирование ПСП

Проект на **C++17**. Реализованы три кастомных генератора псевдослучайных последовательностей (ПСП), статистический анализ, критерий Хи-квадрат, пять NIST-тестов и бенчмарк производительности.

## Структура проекта

| Файл | Описание |
|------|----------|
| `generator.h` | Интерфейс `IGenerator` и реализации: LCG, Von Neumann+Weyl, Xorshift32, `std::mt19937` |
| `tests.h` | Описательная статистика, критерий Хи-квадрат, 5 NIST-тестов |
| `psp_main.cpp` | Точка входа: генерация выборок, тесты, бенчмарк, экспорт CSV |
| `generate_psp_plots.py` | Построение графиков из CSV-файлов |
| `Doxyfile` | Конфигурация Doxygen для API-документации |

## Генераторы

### 1. LCG (линейный конгруэнтный метод)

\[
X_{n+1} = (a \cdot X_n + c) \bmod m
\]

| Параметр | Значение |
|----------|----------|
| a | 1 103 515 245 |
| c | 12 345 |
| m | 2³¹ |

### 2. Von Neumann + Weyl (метод середины квадрата)

Базовый алгоритм: \(X_{n+1} = \text{middle}(X_n^2)\).

**Модификация Вейля** предотвращает вырождение в ноль:

\[
X_{n+1} = \text{middle}(X_n^2 + w_n), \quad w_{n+1} = w_n + \delta
\]

где \(\delta = \texttt{0x9E3779B97F4A7C15}\) (константа Вейля).

### 3. Xorshift32

Классический алгоритм Марсаглии:

```
state ^= state << 13;
state ^= state >> 17;
state ^= state << 5;
```

## Статистический анализ

- **20 выборок** по **1000 элементов** для каждого генератора
- Диапазон значений: **[0, 5000]**
- Для каждой выборки вычисляются:
  - математическое ожидание (среднее);
  - дисперсия (несмещённая, \(n-1\));
  - среднеквадратичное отклонение;
  - коэффициент вариации \(CV = \frac{\sigma}{\mu} \cdot 100\%\).

## Критерий Хи-квадрат

Проверка гипотезы о **равномерном распределении**:

- Диапазон разбивается на **K = 10** карманов
- Степени свободы: \(\nu = K - 1 = 9\)
- Уровень значимости: **α = 0.05**
- Критическое значение: \(\chi^2_{0.05, 9} = 16.919\)

**Вердикт:**
- \(\chi^2 \leq 16.919\) и \(p \geq 0.05\) → гипотеза **ПРИНИМАЕТСЯ**
- иначе → гипотеза **ОТКЛОНЯЕТСЯ**

## NIST-тесты (адаптированные)

| № | Тест | Что проверяет |
|---|------|---------------|
| 1 | Monobit Test | Баланс нулей и единиц в битовой последовательности |
| 2 | Frequency within Block | Частота единиц внутри блоков по 128 бит |
| 3 | Runs Test | Длины серий одинаковых битов |
| 4 | Longest Run of Ones | Самая длинная серия единиц в блоке |
| 5 | Serial Test | Частоты перекрывающихся пар битов |

Гипотеза принимается при **p-value ≥ 0.05**.

## Интерпретация результатов

- **LCG** — быстрый, но слабая битовая структура (Monobit/Runs часто отклоняются)
- **Von Neumann+Weyl** — медленнее, непредсказуемое качество битов
- **Xorshift32** — хороший баланс скорости и качества
- **std::mt19937** — эталон STL, медленнее кастомных LCG/Xorshift

Хи-квадрат проверяет **равномерность в диапазоне [0, 5000]**, NIST-тесты — **битовую случайность**. Расхождение результатов нормально: это разные аспекты качества ПСП.
